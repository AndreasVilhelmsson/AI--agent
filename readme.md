# AI Meeting Assistant (MVP)

A real-time AI assistant that analyzes meeting notes and produces an editable summary and action items.

## Tech Stack
- Frontend: React + TypeScript (Vite)
- Backend: .NET 8 + SignalR (WebSocket)
- AI Integration: LLM function calling (OpenAI)
- Database: SQLite (MVP)
- Deployment: Docker + Azure

See `/docs` folder for full project plan.
